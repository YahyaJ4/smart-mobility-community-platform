# Smart Mobility Community Platform

A web-based platform developed to support community transportation access by connecting riders and drivers through a simple mobility coordination system.

## Overview

This project was developed during my software engineering internship at the Urban Information Lab at the University of Texas at Austin.

The platform was designed as a prototype for the Community Hub for Smart Mobility, helping coordinate transportation requests, rider information, driver views, and map-based pickup locations.

## Features

- Rider registration and information management
- Driver-facing trip and rider views
- Interactive map views using Leaflet.js
- JSON-based data storage prototype
- Simple web interface for mobility coordination

## Technologies Used

- HTML
- CSS
- JavaScript
- Python CGI
- JSON
- Leaflet.js
